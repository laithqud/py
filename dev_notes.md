## Start a new project
git init

git status

git add .

git commit -m "message"

git remote add origin <repo_url>

git branch -M main

git push -u origin main

## Check remote
git remote -v

## Push after first time
git push

## Pull changes from GitHub
git pull origin main

## View commit history
git log

## Exit git log
q
